Plugin temporär einbinden:

1. Link in Adressleiste eingeben: about:debugging#/runtime/this-firefox 
2. Auf Button "Temporäres Add-on laden" klicken
3. Manifest auswählen

Falls der Link nicht funktioniert:
1. Einstellungen öffnen (Logo ganz rechts in Leiste mit drei Strichen)
2. Add-Ons und Themes
3. Auf das Zahnradsymbol klicken -> Add-Ons debuggen
4. Auf Button "Temporäres Add-on laden" klicken
5. Manifest auswählen
